# Background Pitch — Dual Track (Manufacturing vs Test)

Use **only one pitch per call**.  
Pick based on the role family. Do not blend.

---

## Pitch A — Manufacturing Engineer (20 seconds)

**Use for:**  
Manufacturing Engineer, Manufacturing Development, Process, NPI, Production Readiness

> I’m a manufacturing engineer in aerospace and defense focused on high-voltage battery module production. I’ve owned end-to-end build flow including process definition, fixtures and tooling, work instructions, and production readiness, plus the troubleshooting loop when builds fail.  
> At Terran Orbital I stabilized and scaled battery module assembly by tightening process controls and improving repeatability, and I supported assembly-side testing to prevent defects from moving downstream.

Stop.

---

## Pitch B — Hardware Test / Test Automation (20 seconds)

**Use for:**  
Hardware Test Engineer, Electrical Validation, Manufacturing Test, Test Automation (hardware-facing)

> I’m a hardware test and validation engineer with aerospace and defense experience. My core strength is assembly-side test execution—fixtures and harness interfaces, power and comms bring-up, bench debug, and validating test setups so results are trustworthy.  
> At Terran Orbital I supported battery module testing including a 16-channel flashing and telemetry fixture and helped qualify electrical test stations to reduce false fails. I’m also building a Python test runner for per-serial execution and traceability, and I’m adding instrument control and CAN logging.

Stop.

---

## Precision Scope Clarifier (use once if asked)

> To be precise, my ownership was assembly-side test execution and validation support. Final module acceptance before spacecraft integration was owned by a different team.

---

## Fast Decision Rule

- If the job description emphasizes **yield, process, line, tooling, NPI, work instructions** → use **Pitch A**
- If it emphasizes **validation, test stations, automation, instrumentation, debug, protocols** → use **Pitch B**

Do not improvise. Pick one and run it.
